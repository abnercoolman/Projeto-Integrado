# 1. Objetivo

Desenvolver uma plataforma, com foco na gestão de recursos humanos.

### 2. Estruturação do Desenvolvimento da Plataforma

- Páginas:
  - Login;
  - Cadastro;
  - Home;
  - Detalhes do Funcionário;
  - Folha de Pontos;
  - Perfil.
